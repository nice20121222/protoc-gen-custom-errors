**Goal:** Reproduce the source error generator in this repository while replacing all legacy branding with `protoc-gen-custom-errors`.

**Architecture:** Keep the source layout: a `main` package drives `protogen`, `errors/errors.proto` defines enum extensions, and an embedded Go template emits Kratos error helpers. **Tech Stack:** Go 1.21, `google.golang.org/protobuf`, `golang.org/x/text`, Go templates, Buf v2, Kratos generated-code API.
